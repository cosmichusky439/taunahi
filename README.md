## What is Taunahi?

**Taunahi** is a premium Hypixel Skyblock scripting client for Minecraft 1.21.x, built to automate farming, combat, foraging, fishing, and more — with unmatched safety, realism, and polish. It's the #1 most purchased Skyblock script on the market.

Whether you're grinding coins, leveling skills, or just want to AFK without babysitting your screen, Taunahi handles it.

## Crucial Features

### Hydra Anti-Staff System
The most advanced anti-staff detection in any Skyblock client. Custom-built system called **Hydra** with maximum check detections, hyper-realistic responses, and instant Discord alerts.

### Scripting Sessions & Dashboard
Every scripting session is tracked — earnings, failsafes, screenshots, hourly rates — all visible from your personal dashboard.

### Discord Integration
Live notifications, screenshots, and hourly earnings delivered directly to your private Discord channel.

### Remote Control
Control your active script from the dashboard or Discord — no need to be at your PC.

### Humanized Movement
Movement and rotations based on real inputs like mouse sensitivity, making it reverse-engineering resistant.

### Dynamic Rest
Set a realistic sleep cycle to simulate legit activity patterns and stay under the radar.

### Picture-in-Picture Mode
Always-on-top mini window — keep an eye on your script while doing something else.

### Auto-Sell
Automatically sells to NPC or Bazaar, whichever is more profitable at the time.

### Instant Anti-Stuck
Reliable block-escape with no scuffed wiggling.

### Failsafe
Handles hub switches, disconnects, and reconnects reliably. Supports "Return to Location" on failsafe rejoin.

## Getting Started

### Installation

1. Download the latest `.jar` from the [Releases](../../releases/latest) tab above
2. Install [Fabric Loader](https://fabricmc.net/use/installer/) for Minecraft 1.21.x
3. Install [Fabric API](https://modrinth.com/mod/fabric-api) — drop the `.jar` into your `mods` folder
4. Drop the Taunahi `.jar` into your `.minecraft/mods` folder
5. Launch Minecraft using the Fabric 1.21.x profile
